# ⚖️ Law Chambers Website — Adv. Nidhi Sharma

A modern, responsive, and professional legal website built using React, designed to represent the practice of **Adv. Nidhi Sharma** with clarity, credibility, and elegance.

---

## 🌐 Live Website
🔗 https://your-vercel-link.vercel.app

---

## 📌 Overview

This website serves as a digital presence for a legal practice, showcasing:

- Advocate profile
- Practice areas
- Contact information
- Direct consultation options (WhatsApp, Email, LinkedIn)
- Office location (Google Maps)

The design follows a **minimal, premium, law-firm aesthetic** using a black and gold theme.

---

## ✨ Features

- ⚡ Fully responsive design (mobile + desktop)
- 🎯 Smooth scroll navigation
- 💼 Professional UI/UX (law firm standard)
- 📱 WhatsApp consultation integration
- 🔗 LinkedIn & Email connectivity
- 📍 Embedded Google Maps location
- 🟡 Elegant gold-accent design system

---

## 🛠️ Tech Stack

- React.js
- CSS3 (Custom styling)
- React Icons
- Vite (Build tool)
- Vercel (Deployment)
